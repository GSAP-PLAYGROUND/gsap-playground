# Security Policy

## Supported Versions

Currently, the following versions of the playground receive security updates:

| Version | Supported |
| ------- | --------- |
| v0.1.x  | ✅ Yes     |
| < v0.1  | ❌ No      |

---

## Reporting a Vulnerability

We take the security of this playground seriously. Since this is an interactive sandbox website, vulnerabilities might include cross-site scripting (XSS), packaging issues, or dependencies risks.

If you find a security vulnerability, please do **not** open a public issue. Instead, report it using one of the following methods:

1. **GitHub Security Advisories**: If available, please submit a report privately via the "Security" tab on GitHub.
2. **Private Email**: Email the repository owner (Ritesh Sinha) directly to report the findings.

Please include:
*   A clear description of the vulnerability and its potential impact.
*   Step-by-step instructions (with proof-of-concept code if possible) to reproduce the behavior.

We will acknowledge receipt of your report within 48 hours and work with you to resolve it promptly before publishing a fix. Thank you for helping keep our community safe!
